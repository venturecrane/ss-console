# Session Notes

Raw session notes from each decision-layer working session. One file per session, numbered sequentially.

## Naming Convention

```
NNN-layer-topic.md
```

Examples:

- `001-buybox-vertical.md`
- `002-default-stack-scope.md`
- `003-pricing-payment.md`

## What Goes Here

- Raw reasoning, tradeoffs discussed, options rejected and why
- Links to the decision issues each session informed
- Timestamped, not polished — capture the thinking, not a summary

## What Doesn't Go Here

- Final decisions (those go in `PACKAGE-SPEC.md`)
- Action items (those go in GitHub issues)
