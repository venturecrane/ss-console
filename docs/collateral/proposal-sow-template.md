## Internal: Filling Out This Template

**After the assessment call and solution design:**

1. Pull findings from the MacWhisper extraction / capture doc
2. "What We Heard" тАФ reflect their objectives and problems using their language
3. "What We'll Build" тАФ translate the solution design into client-facing deliverables (no internal jargon, no tool names unless discussed on the call)
4. "What's Not Included" тАФ list anything adjacent that could cause scope confusion
5. Project price = estimated hours x current rate tier, presented as a flat number
6. Choose 50/50 or 40/30/30 payment structure based on estimated hours
7. Set tentative start date based on current capacity
8. Export to PDF, add signature fields, send via DocuSign or Dropbox Sign

**Tone:** Direct, clear, no fluff. The owner should be able to read this in 5 minutes and know exactly what they're getting, what it costs, and what happens next. No sales language тАФ the assessment call already did that work.

**Reminder:** No hourly breakdown. No rate disclosure. Project price only.
